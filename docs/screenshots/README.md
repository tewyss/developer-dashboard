# Screenshots

Add PNG screenshots here so they show up in the main `README.md`. Expected filenames:

| File | Page to capture |
|------|-----------------|
| `home.png` | Public home / hero page (`/`) |
| `projects.png` | Public projects list (`/projects`) |
| `admin-dashboard.png` | Admin dashboard with stat cards (`/admin`) |
| `admin-projects.png` | Admin project list or create/edit form (`/admin/projects`) |
| `admin-blog.png` | Admin blog list (`/admin/blog`) |
| `admin-todo-notes.png` | TODO board and/or Notes (`/admin/todos`, `/admin/notes`) |

Tip: capture at a consistent width (e.g. 1280px) for a tidy README. On Windows, `Win + Shift + S` opens the snipping tool.
